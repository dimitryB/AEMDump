# Code of Conduct

This project is committed to a respectful and inclusive community.

## Our Standards

- Be respectful and constructive.
- Assume good intent and discuss ideas, not people.
- Avoid harassment, discrimination, or abusive behavior.

## Scope

This Code of Conduct applies to project spaces, including issues, pull requests, and discussions.

## Enforcement

Project maintainers may remove comments, reject contributions, or block participation for behavior that violates this policy.
